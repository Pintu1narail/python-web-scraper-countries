# python-web-scraper-countries
"A Python-based web scraper that extracts country details including capital, population, and area from ScrapeThisSite using BeautifulSoup and Requests."

🌍 CountryData-Scraper
This project is a Python web scraper that collects basic information about countries from scrapethissite.com. It is an excellent example for beginners to learn the BeautifulSoup and Requests libraries.
🚀 Features
Scrapes the names of all countries worldwide.
Collects each country's Capital, Population, and Area (sq km).
Automatically saves the collected data into a country_info.csv file.
Clean and structured code.
🛠️ Tech Stack
Python 3.x
BeautifulSoup4 (for HTML parsing)
Requests (for sending HTTP requests)
CSV Module (for data storage)
